APIron Lab – LLM ユーティリティ API

LLM ワークフローのための 実務寄りユーティリティAPI：文字化け修復、トークンカウント、事前入力整形。

EN / 日本語

日本語および多言語テキストの処理に特化したユーティリティ API 群。

APIron Lab は、LLM やデータ処理ワークフローを下支えする 「実務寄り・ユーティリティ系 API」 を開発・公開しています。

文字化け修復やトークン数計測など、 LLM に渡す前の事前処理（Pre-AI Input Hygiene） に特化した API を中心に展開します。

各 API は GitHub 上のオープンコア と RapidAPI 上のホスティング版 の両方から利用できます。

公開中の API

Encoding / Text Hygiene

Encoding Repair API (v2.0)

UTF-8 / Shift_JIS / EUC-JP / Latin-1 などをまたいだ 文字化け（mojibake）を修復 する Base64 専用エンコーディング修復 API。

  • Base64 生バイト入力専用で情報欠損を防止
  • 自動エンコーディング判定 & セーフフィルタ
  • UTF-8 への統一出力で下流処理を簡素化
  • 日本語・多言語テキストパイプラインを想定
Docs (GitHub Pages / EN) · GitHub · RapidAPI
Token / LLM Utility

Universal Token Counter (UTC)

OpenAI 互換エンコーディング向けの高精度トークンカウンタ API。 統一された JSON レスポンスで、コスト試算やログ集計に最適。

  • OpenAI 互換トークナイザ（o200k_base / cl100k_base）に対応
  • 日本語を含む多言語テキストを前提とした設計
  • result + meta の安定したレスポンススキーマ
  • コスト試算・モニタリング・ログ基盤向けの軽量 API
Docs (GitHub Pages / EN) · GitHub · RapidAPI
